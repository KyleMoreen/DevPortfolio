Student Randomizer
A browser-based application that takes in a list of students from a local JSON file and selects a random student. The application dynamically generates buttons for different classes, keeps track of which students have already been picked, shows how many students are left in the list, and resets the list automatically when all students have been selected. This project was designed using HTML, CSS, and Javascript.

Prisoner's Dilemma: The Game
A game based on the concept of the Prisoner's Dilemma. Each turn, the player must decide to either take money from the pot or leave it. If only one player decides to take the money, it is theirs. If both pass, the pot doubles. If both players try to take the money, both players lose everything. This project was designed using HTML, CSS, and Javascript.

Pluto Demotion Timer
A counter that shows the exact amount of time since Pluto was declared a dwarf planet. The counter updates in real time. Also features a contact form for those who would like to petition for Pluto's reinstatement. This project was designed using HTML, CSS, and Javascript.
